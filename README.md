# Compilador (AA)   
Este es un proyecto de compilador en desarrollo para la asignatura de Análisis de Algoritmos (AA).

⚠️ **Estado actual:** Pre-alpha / Básico.

## Descripción
Actualmente, el compilador cuenta con una estructura mínima. El objetivo es ir implementando progresivamente las distintas fases de compilación:

- [ ] Análisis Léxico (Scanner)
- [ ] Análisis Sintáctico (Parser)
- [ ] Análisis Semántico
- [ ] Generación de Código Intermedio
- [ ] Optimización
- [ ] Generación de Código Final
